# AI-Use Statement (optional — insert only if the target journal requires or permits it)

**Status: NOT currently in the manuscript.** This is a suggested, ready-to-use
statement. Decide based on the target student journal's policy:

- If the journal **requires** an AI-use / author-contribution transparency statement:
  paste the text below into the manuscript, typically as a short subsection near
  "Author contributions" or "Acknowledgements."
- If the journal is **silent** on AI use: adding a brief, honest statement is
  generally prudent and low-risk.
- If the journal **forbids** AI assistance: do **not** add this statement, and
  ensure the submission complies with that policy.

Do not add this statement solely to acknowledge AI; add it only where transparency
about tool use is expected or beneficial.
